# GitHubサンプル

GitHubのサンプルサイトです．

## 機能
 
* YOLOv8を用いて，サンプル画像の骨格情報を出力します．
  
## インストール
* Anacondaがインストールされていることが前提条件です．
* 以下のPythonモジュールが必要です．
```bash
pip install ultralystic
```

## 利用法
 
* 以下のように実行します．
```bash
python pose.py
```
