# GitHubサンプル

GitHubのサンプルサイトです．

## 機能
 
* YOLOv8を用いて，サンプル画像の骨格情報を出力します．
  
## インストール法
* Anacondaがインストールされていることが前提条件です．
* リポジトリをクローンします．
```bash
git clone https://github.com/kitamura-lab/GitHubSample.git
```
* パッケージをインストールします．
```bash
pip install ultralytics
```

## 利用法
 
* プログラムを実行します．
```bash
python pose.py
```
